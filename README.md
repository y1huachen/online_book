# a springboot project
## info
模仿起点中文网的在线阅读项目，采用SpringBoot+JPA+MySQL构建后端存储与逻辑，thymeleaf作为页面模板，JQuery+LayUI构建前端界面，大部分交互数据（如分类、书籍、目录、章节等）采用ajax异步获取，前台实现分类展示、在线阅读、修改头像、书籍下载、留言互动等基础功能，后台实现基础信息管理。
